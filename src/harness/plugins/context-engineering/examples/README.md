# Orchestration Examples

Practical implementation examples for multi-agent orchestration patterns.

## Examples

| Example | Pattern | Use Case |
|---------|---------|----------|
| [code-review-system](./code-review-system.md) | Hierarchical + Parallel | Automated code analysis with specialized agents |
| [research-assistant](./research-assistant.md) | Pipeline + Verification | Multi-source research with fact-checking |
| [content-pipeline](./content-pipeline.md) | Sequential + Parallel | Content generation with quality gates |

## Pattern Mapping

These examples demonstrate key patterns from `specs/orchestration-patterns/`:

- **Hierarchical Coordination**: Coordinator delegates to specialists
- **Parallel Execution**: Independent tasks run concurrently
- **Pipeline Pattern**: Sequential stages with quality gates
- **Synthesis Pattern**: Aggregating results from multiple agents

## Usage

Each example includes:
1. Architecture diagram
2. Implementation code
3. Optimization notes
4. Integration points with harness infrastructure
