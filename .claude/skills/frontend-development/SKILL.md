# Frontend Development Skill

## Instructions

When working with frontend development tasks:

1. **Component Design**: Create reusable, accessible components
2. **State Management**: Choose appropriate state solutions
3. **Styling**: Use modern CSS patterns and responsive design
4. **Performance**: Optimize bundle size and runtime performance
5. **Testing**: Write unit and integration tests

## Templates

- [React TypeScript App](./templates/react-typescript-app.md) - Full React application setup

## Guidelines

- Follow React best practices and hooks patterns
- Ensure accessibility (WCAG 2.1 AA)
- Implement responsive design
- Optimize for Core Web Vitals
- Use TypeScript for type safety
