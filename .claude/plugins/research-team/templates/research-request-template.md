# Research Request Examples

This document provides flexible examples of research requests that activate the research-team plugin. These are reference examples only - actual requests can vary widely.

## Basic Research Request

```
Research the latest developments in quantum computing
```

**What happens:**
- Lead coordinator breaks into 4 subtopics
- Spawns parallel researchers for each angle
- Synthesizes findings into comprehensive report

## Market Research

```
Do a competitive analysis of electric vehicle manufacturers
```

**Focus areas might include:**
- Market share and growth trends
- Technology differentiation
- Major players and positioning
- Future outlook and projections

## Technical Comparison

```
Research Python web frameworks - I need to choose one for my project
```

**Typical coverage:**
- Feature comparison (Django, Flask, FastAPI, etc.)
- Performance characteristics
- Ecosystem and community
- Use case recommendations

## Industry Analysis

```
Research the AI chip market and major players
```

**Common subtopics:**
- Market size and growth
- Key companies (NVIDIA, AMD, Intel, startups)
- Technology trends
- Supply chain considerations

## With Formatting Preference

```
Research neural architecture search for my Joplin notes
```

**Special handling:**
- Activates joplin-research skill
- Outputs markdown with proper formatting
- Optimized for note-taking application

## Notes

- Research requests can be any length from one sentence to detailed briefs
- More specific requests lead to more targeted research
- Mentioning "Joplin" ensures markdown formatting
- The coordinator adapts to request complexity (2-4 subtopics)
